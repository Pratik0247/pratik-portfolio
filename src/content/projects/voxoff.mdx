title: "VoxOff — End-to-end Karaoke Generator"
slug: "voxoff"
summary: |
  Developed an end-to-end karaoke generation system combining Spleeter for vocal
  separation and ForceAlign for lyrics synchronization. Deployed as microservices
  on GKE with RabbitMQ orchestration and Terraform automation. Integrated OAuth,
  Firestore, and GCS for a seamless hybrid local/cloud pipeline, processing
  100+ live requests with ~2–3 min average per song.
stack: [GKE, RabbitMQ, Spleeter, ForceAlign, Terraform, GCS, OAuth, Firestore]
