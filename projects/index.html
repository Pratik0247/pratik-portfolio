Projects

Case studies of systems, infra, and AI projects.

MoodyTunes — Deep Learning Music Mood Classifier

Built a deep learning-based system to classify songs into moods (happy, sad, calm, energetic). Enriched Spotify’s 278k-song dataset with Last.fm tags, engineered features, and trained multiple architectures (ANN, RCNN, LSTM), achieving 90.67% accuracy while reducing 7k+ tags to 35 core categories.

NarrateNow — Cloud-based Chapter-wise Audiobook Generator

Implemented a cloud-native service to convert EPUB books into chapter-specific audiobooks using Google TTS. Designed a Kubernetes microservice pipeline with RabbitMQ, Redis, and GCS for storage. Processed an 11-chapter, 150-page book in under 4 minutes, scaling to 50+ concurrent requests with real-time progress tracking and automated recovery.

ResMatch — Resume Analyzer & Job Recommender

Built a resume analysis and job recommender platform using Sentence-BERT and scikit-learn for semantic matching. Backend in Flask, deployed via Docker on GCP Cloud Run with CI/CD using GitHub Actions. Implemented daily job alerts through Cloud Scheduler and Pub/Sub pipelines for automated notifications.

Structured Reasoning for Table QA

Designed a two-phase prompting pipeline for table question answering: (1) column selection using LLM reasoning, followed by (2) answer generation restricted to those columns. Incorporated FAISS retrieval and LLaMA-3 models. Achieved 87% column accuracy and 49.7% answer accuracy on the DataBench dataset.

VoxOff — End-to-end Karaoke Generator

Developed an end-to-end karaoke generation system combining Spleeter for vocal separation and ForceAlign for lyrics synchronization. Deployed as microservices on GKE with RabbitMQ orchestration and Terraform automation. Integrated OAuth, Firestore, and GCS for a seamless hybrid local/cloud pipeline, processing 100+ live requests with ~2–3 min average per song.
