Structured Reasoning for Table QA

Designed a two-phase prompting pipeline for table question answering: (1) column selection using LLM reasoning, followed by (2) answer generation restricted to those columns. Incorporated FAISS retrieval and LLaMA-3 models. Achieved 87% column accuracy and 49.7% answer accuracy on the DataBench dataset.
